# 02_MVP_FEATURES.md

## MVP objective
Launch the first usable foundation of UyTop with real architecture and the most important product flows.

## MVP must-have features

### Authentication
- register
- login
- JWT auth
- refresh token flow
- profile basics
- role-aware access structure

### Listings
- create listing
- edit listing
- delete listing
- save draft
- submit for moderation
- list/detail endpoints
- owner listing management

### Map discovery
- map + list hybrid architecture
- map bounds search endpoint
- listing preview data for markers
- clustering-ready backend design

### Search and filters
- sale/rent
- city
- district
- category
- price range
- rooms
- size
- sort options
- featured and verified filters

### User features
- favorites
- recently viewed
- saved searches
- notifications foundation

### Moderation
- admin approve/reject flow
- fake listing reports
- moderation status model
- moderation action log

### Owner tools
- my listings
- listing status tracking
- analytics foundation
- sold/rented status

### Admin panel
- login-protected admin shell
- moderation queue
- reports page
- users/owners basic management
- anti-fraud dashboard foundation

## MVP technical must-haves
- Docker/dev environment
- `.env.example` files
- OpenAPI starter
- clean monorepo structure
- PostgreSQL + PostGIS config
- Redis + Celery wiring
- test foundations

## Post-MVP features
- search alerts
- owner verification flow
- duplicate listing detection
- mortgage calculator UI polishing
- richer analytics
- subscription/featured purchase workflow
- public SEO web frontend
