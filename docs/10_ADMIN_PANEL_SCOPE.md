# 10_ADMIN_PANEL_SCOPE.md

## Initial Admin Surfaces
- dashboard
- pending listing moderation queue
- reported listings list
- users and owners basic management

## Later Admin Surfaces
- owner verification management
- featured promotions
- subscriptions
- anti-fraud dashboard
- audit log viewer

## Backend Dependencies
Admin workflows depend on:
- role-based auth
- pending listing endpoints
- report list endpoints
- moderation approve/reject endpoints
- audit log records
