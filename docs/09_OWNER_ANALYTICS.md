# 09_OWNER_ANALYTICS.md

## Initial Analytics Inputs
- listing view count
- favorite count
- contact click counts by channel
- featured flag and expiry
- lifecycle outcome: active, sold, rented, archived

## Data Sources
- `Listing.view_count`
- `Favorite`
- `ContactClickLog`
- listing lifecycle timestamps

## MVP Output
Analytics remains a later phase feature, but current data models should support:
- total views per listing
- total favorites per listing
- contact clicks by channel
- listing status conversion summary
