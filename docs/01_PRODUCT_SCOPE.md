# 01_PRODUCT_SCOPE.md

## Product name
UyTop

## Product summary
UyTop is a real estate marketplace for discovering houses, apartments, land, and commercial property through search, filters, and an interactive map.

## Core users
1. Guest
2. Buyer / Renter
3. Property Owner
4. Admin

## Core user goals
### Guest
- browse listings
- search and filter
- view details
- register/login

### Buyer / Renter
- save favorites
- view recently viewed
- save searches and alerts
- report suspicious listings
- contact owners

### Property Owner
- create and manage listings
- upload images
- save draft listings
- submit listings for moderation
- mark sold/rented
- track listing performance

### Admin
- approve/reject listings
- review reports
- detect suspicious content
- manage owners, users, categories, locations, promotions, subscriptions
- review anti-fraud metrics

## Property scope
Supported categories:
- House
- Apartment
- Land
- Commercial property

Listing types:
- Sale
- Rent

## Core product pillars
1. Map-based discovery
2. Real listing management for owners
3. Moderation and anti-fraud
4. User retention features
5. Owner analytics
6. Future monetization readiness
7. Multilingual-ready architecture

## Required platforms
- Flutter mobile app
- Django + DRF backend
- Next.js admin panel

## Non-functional goals
- production-ready structure
- modular codebase
- security-first backend
- maintainable architecture
- scalable data model
- fast map and search experience
- clean API contracts

## Out of scope for initial phase
- full payment gateway integration
- public SEO web frontend
- advanced recommendation engine
- complex messaging platform
- enterprise BI/reporting

## Phase 1 target
Deliver a strong technical foundation that future contributors can extend safely.
